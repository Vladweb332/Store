Инструкция по запуску проекта
Шаг 1. Подготовка рабочего окружения
Убедитесь, что у вас установлен текстовый редактор (например, Visual Studio Code) и веб-браузер.
Скачайте файлы проекта, если они находятся в архиве, и распакуйте их в отдельную папку.


Шаг 2. Установка и настройка
Если используется TailwindCSS (как в данном проекте):

Убедитесь, что на компьютере установлен Node.js.
Установите TailwindCSS:
bash
Копировать код
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init
Настройте tailwind.config.js для включения всех классов:
javascript
Копировать код
module.exports = {
  content: ["./*.html"],
  theme: {
    extend: {},
  },
  plugins: [],
};
Включите Tailwind в CSS-файл (например, styles.css):
css
Копировать код
@tailwind base;
@tailwind components;
@tailwind utilities;
Если Tailwind не используется (статический проект), то файлы уже настроены, и дополнительные шаги не нужны.


Шаг 3. Запуск проекта
Откройте файл index.html в веб-браузере:

Если Tailwind подключен, соберите стили:
bash
Копировать код
npx tailwindcss -i ./input.css -o ./output.css --watch
Перейдите в папку проекта и дважды кликните по index.html, чтобы открыть его в браузере.
Убедитесь, что все элементы отображаются корректно:

Навигационная панель (вверху).
Боковое меню (открывается при нажатии на кнопку ☰).
Сетка товаров (подгружает товары из массива).


Шаг 4. Тестирование функций
Нажмите на кнопку ☰ в правом верхнем углу, чтобы проверить работу бокового меню.
Проверьте поиск товаров в боковом меню.
Убедитесь, что категории товаров отображаются корректно.
Проверьте работу ссылок на товары (если добавлены заглушки).


Шаг 5. Дополнительные действия (если необходимо)
Для изменения списка товаров откройте файл script.js и добавьте новые элементы в массив.
Для изменения категорий бокового меню измените содержимое <ul> в блоке sidebar.
